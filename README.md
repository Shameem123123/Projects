# Quiz App 🎯

A modern and interactive online quiz application built using Django.  
This project allows users to attempt quizzes, navigate through questions, skip questions, view final scores, and review answers with a clean and responsive UI.

---

## Features

- Interactive quiz interface
- Multiple-choice questions
- Question skipping support
- Score calculation system
- Review section after quiz completion
- Correct and wrong answer highlighting
- Skipped question identification
- Smooth scrolling review navigation
- Responsive modern UI
- Animated transitions and clean card layouts
